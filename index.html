High Performance Data Platform

We challenged conventions to bring you the most mechanically sympathetic database software there is. Performance that redefines what's possible. Accessibility without a compromise. We want you to love working with data again.


Watch video

QuestDB is a column-based database optimised for time series data. We extended familiar SQL to work well with time series and temporal data. We provided intelligent REST API for data import and export to JSON and text formats. We built Web UI for effortless data access.

Memory mapped storage

QuestDB does not create copies of data thus increasing IO performance. Written data is immediately available across all processes. There is no data loss should process exit abnormally.

Column based design

Only data that used in query is read from disk.

Partitioned data

Data can be partitioned by time. Time-based queries take advantage of partitions reducing IO.

Bespoke SQL optimiser

We believe queries should run fast, all of the time. It is optimiser's job and QuestDB optimiser has it covered. We made our optimiser establish table join order, derive implied dependencies, remove constant predicates, add transitive predicates, move predicates inside of sub-queries, remove unnecessary ordering, remove unnecessary sub-queries and much more

Rich SQL experience

QuestDB supports almost entire array of standard SQL features. This includes joins of any number of tables, inner joins, outer joins and cross joins. We built support for sub-queries, aggregation, filtering, grouping, ordering, analytic functions. We extended SQL to support time series with "as of" join, re-sampling and flexible time interval search.

REST API

Query, data import and export API

Web Console

We built desktop-like user experience in browser with our bespoke JS grid system

Bespoke HTTP Server

Our HTTP server is built to handle massive amount of data. It is based on native non-blocking IO and supports flow control

QuestDB 3.0 64-bit

Performance, Distilled
QuestDB supports 64-bit versions of Windows, Linux and OSX. Distribution package is cross-platform and does not require installation. Just unzip and run.
